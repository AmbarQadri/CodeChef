Git & GitHub Workshop

The recent Git and GitHub workshop, conducted by Mohd. Ambar Qadri and Adil Ahmad, was a resounding success. Participants learned essential version control skills and best practices for collaborative software development. The workshop included hands-on exercises with Git commands and an in-depth exploration of GitHub functionalities. Attendees appreciated the interactive format and clarity of the presentations. We thank everyone involved for making this event a valuable learning experience!

Venue: Seminar Hall & Lab 4, CO Department

Time: 2:30 PM - 4:30 PM

Speaker: Mohd. Ambar Qadri, Adil Ahmad
